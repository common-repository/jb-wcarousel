== Installation ==

1. Upload 'jb-wCarousel' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. you'll see a side bar named wCarouse , drop widget that you want to be carousled
4. drop the JB-wCarousel widget to the main sidebar for the carousel output
